# File Sharing System

## Introduction

This is a secure file-sharing system between two different types of users: Ops Users and Client Users.

## Features

- **Ops User:**
  - Login
  - Upload File (only pptx, docx, and xlsx)

- **Client User:**
  - Sign Up (returns an encrypted URL)
  - Email Verification
  - Login
  - Download File
  - List All Uploaded Files
